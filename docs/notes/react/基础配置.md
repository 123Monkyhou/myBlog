react是mvc模式,vue是mvvm
    mvc:
    @1 需要构建试图,通过jsx
    @2 构建数据层:但凡在视图中需要动态处理的都需要有对应的数据模型
    @3 控制层:当在视图中进行某些操作时,都是去修改相关的数据,react按照最新的数据去渲染
    "数据驱动试图的渲染"
    视图中的表单改变,需要开发者自己去写代码实现修改数据
    mvvm:
    不仅监听数据的改变,也能监听到视图的改变
    "双向驱动"
